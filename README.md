## ✨ What is this?

**Contact Book App** is a Python terminal application that lets you manage contacts with full **CRUD operations** — no database, no framework, just pure Python logic!

---

## ⚡ Features

- ✅ **Create Contact** — add name, age, email, phone number
- ✅ **View Contact** — search and display contact details
- ✅ **Update Contact** — modify existing contact information
- ✅ **Delete Contact** — remove a contact permanently
- ✅ **Count Contacts** — see total number of saved contacts
- ✅ **Duplicate Prevention** — alerts if contact already exists
- ✅ **Interactive Menu** — clean numbered menu interface
- ✅ **Error Handling** — handles invalid inputs gracefully
